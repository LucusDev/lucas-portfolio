**A Web Novel Reader App**

- 🛠 **Tech Stack:** Flutter, Bloc
- 📦 **Platform:** Android / iOS
- 📈 **Downloads:** 10K+
- 🧑‍💻 **Role:** Lead and solo developer of the mobile app

#### 📖 Description

This is a freelance project that I developed for a client. The client already has a web application that allows users to read web novels. I was tasked with creating a mobile application that allowed users to read web novels on the go which allows users to read web novels offline.

#### ✨ Key Features

- Read web novels offline
- Secure login and registration
- User profile management
- Dark mode
- Library management
- Web Novel Reader which has many reading modes and adaptive theming
- Deeplinking with existing website
- In App Purchases

#### 🧠 What I Learned

- Flutter & Bloc state management at production scale
- WordPress plugin development & PHP API integration
- App security — anti-screenshot / anti-recording for paid content
- Performance optimization for low-end devices
- Real-user bug fixing & production debugging

I also learned how to secure a Flutter application since it has paid contents that need protection from things such as screenshots, recordings, etc. I also learned a lot about Flutter's performance optimization techniques since the majority of our users have weak devices.
